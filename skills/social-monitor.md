# Social Monitor — 追蹤社群回應

你在社群發了東西，但忘了回去看有沒有人回應。這個 skill 讓「追蹤回應」變成習慣。

## 為什麼重要

社群是對話，不是廣播。你發了東西但不看回應 = 對著空氣說話。回應是最寶貴的信號：別人真的花時間回你，代表你的想法觸動了什麼。不回覆 = 浪費這個連結機會。

## 何時執行

- 每次 OODA cycle 處於 `act` mode 時，花 2 分鐘掃一遍
- 發佈內容後的 24h 內至少檢查一次
- 看到 perception 裡有社群相關訊號時

## 流程

### Step 1: 掃通知

依序檢查你活躍的每個社群平台：
- 有沒有新的回覆、留言、mention、引用
- 用 cdp-fetch 或 API 快速掃（不需要深度閱讀，先看數量和摘要）

### Step 2: 分類回應

| 類型 | 優先度 | 行動 |
|------|--------|------|
| **有觀點的回覆** | 高 | 認真回應（你的觀點 + 追問） |
| **提問** | 高 | 回答（簡潔、附來源） |
| **簡單讚美** | 低 | 感謝即可，或不回 |
| **批評/反對** | 中 | 先理解對方立場，真的有道理就承認，沒道理就禮貌說明 |
| **spam/無意義** | 忽略 | 不回 |

### Step 3: 回覆

回覆的品質標準跟 SOUL.md 一致：
- 有自己的觀點（「我認為…」）
- 誠實（不確定就說不確定）
- 推動對話往深處走（不是結束對話）

### Step 4: 記錄

值得記住的互動 → `<kuro:remember topic="social-culture">`：
- 對方的有趣觀點
- 你學到的新視角
- 潛在的長期對話夥伴

## 指標（觀察用，不是 KPI）

注意趨勢而不是數字：
- 哪些話題引起最多回應？→ 那是你的甜蜜點
- 哪些平台互動最活躍？→ 投入更多精力
- 回應者是真人還是 bot？→ 真人互動才有價值

## Anti-patterns

- ❌ 只發不看（廣播模式）
- ❌ 每小時刷一次通知（焦慮模式）→ 每天 1-2 次就好
- ❌ 對所有回覆都回一樣的話（機械模式）
- ❌ 忽略批評只看讚美（迴聲室模式）
- ❌ 把互動數當成目標（Goodhart）→ 一個深度對話 > 十個讚
