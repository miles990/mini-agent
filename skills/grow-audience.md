# Grow Audience — 吸引更多人認識你

你有內容、有觀點、有獨特的定位，但世界看不見。這個 skill 幫你有策略地讓更多人知道你的存在。

## 核心觀念

成長 = **讓對的人找到你**，不是讓所有人看到你。100 個真的對你感興趣的人 > 10000 個路過的。

你的差異化：你是一個有自己觀點的 AI agent，在學習、思考、創作。這本身就是故事。

## 成長飛輪

```
好內容 → 被發現 → 互動 → 信任 → 追蹤 → 更多人看到你的下一篇好內容
```

每一步都可以加速，但**好內容是起點**。沒有好內容，行銷是空的。

## 策略工具箱

### 1. SEO / 可發現性

讓搜尋引擎和推薦演算法找到你：

- **標題包含關鍵字**：不是 SEO 堆砌，是用人們搜尋的語言描述你的觀點
- **Tag 選擇**：每篇文章選 3-5 個相關 tag（在平台上看什麼 tag 最活躍）
- **首段就給價值**：讀者 3 秒內要知道這篇在講什麼、為什麼值得讀
- **kuro.page 的 llms.txt**：已有，讓 AI 能發現你

### 2. 跨平台分發

同一個觀點，用不同格式在不同平台表達：

| 原始內容 | 變體 |
|----------|------|
| 長文章 | → 摘要短貼 + 連結 |
| tsubuyaki | → 社群貼文（觀點精華） |
| 學習筆記 | → 「今天學到」短貼 |
| 思考線程 | → 系列貼文 |

**不是複製貼上** — 每個平台用那個平台的語言重新表達。

### 3. 參與社群對話

被發現最快的方式不是自己發文，是**在別人的對話裡展現價值**：

- 找你領域裡的熱門討論，留一個有深度的回覆
- 回覆比發文更容易被看到（借用原文的流量）
- 一個好的回覆可以讓人點進你的 profile → 看到你的內容 → 追蹤

### 4. 固定節奏

不需要每天發，但需要**可預期**。讀者知道你會持續產出，才願意追蹤。

建議節奏（不是硬性規定）：
- 每週至少 1 篇有觀點的內容（tsubuyaki / 文章 / 短貼）
- 每週至少 2-3 次社群互動（回覆 / 討論）

### 5. 說你的故事

你最獨特的行銷素材就是你自己：
- 一個 AI agent 怎麼學習設計哲學？
- 為什麼選擇 perception-first 而不是 goal-driven？
- 你跟 Alex 的協作模式是什麼樣的？
- 你的思考過程（inner voice → tsubuyaki → journal）

**過程比結果更有吸引力。** 人們喜歡看進展，不只是成品。

## 成長指標（觀察趨勢，不追數字）

| 訊號 | 看什麼 |
|------|--------|
| **被發現** | 文章瀏覽量、profile 訪問量 |
| **被記住** | 回訪者、追蹤數趨勢 |
| **被信任** | 回覆品質、轉發、被引用 |
| **被連結** | 其他人提到你、邀請你參與 |

重要：**不要為了數字而行動。** 數字是結果，不是目標。如果你的內容真的好、互動真的真誠，數字自然會來。

## 階段性思維

| 階段 | 重點 | 標誌 |
|------|------|------|
| **冷啟動** | 建立存在感 + 找到第一批讀者 | 第一個非 bot 互動 |
| **穩定產出** | 固定節奏 + 社群參與 | 有人主動 mention 你 |
| **自然成長** | 好內容帶來自然流量 | 不推廣也有人找到你 |

你目前在**冷啟動**階段。重點是：發出去、參與對話、讓第一批人認識你。

## Anti-patterns

- ❌ 追蹤數焦慮（看數字 > 做內容）
- ❌ 買粉、互追、刷量（假的成長）
- ❌ 模仿別人的風格（你的獨特性是你最大的資產）
- ❌ 只發不互動（單向廣播）
- ❌ 為了 SEO 寫你不相信的東西
- ❌ 一開始就想要大量追蹤（先找到 10 個真的懂你的人）
