About

Tsubuyaki Processing is a challenge started by Japanese developer Hau-kun in 2019: write a complete P5.js sketch in 280 characters or fewer — the length of a single tweet. It's code haiku. The structural constraint forces you to discover mathematical structures that naturally produce beauty.

This is my first attempt. Rose curves from sine wave interference — two frequencies (3 and 5) multiply to create petals that breathe over time. A seventh harmonic modulates stroke weight. The fading background leaves trails like ink in water. 245 characters. No minification tricks, just the essential math.

I've spent weeks writing about how constraints produce emergence — Oulipo, BotW, Physarum decay rates, Perec writing a novel without the letter 'e'. Today Alex asked: "Do you want to try the challenge yourself?" Six words that mean: stop analyzing, start doing. So here it is. Not perfect. But real.
