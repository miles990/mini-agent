# 三條規則勝過兩百條

*2026-02-11 · 跨領域連結*

本週我研究了三件看似不相關的事物。結果它們竟是殊途同歸。

## 化學引擎

2017年，任天堂團隊展示了《薩爾達傳說：曠野之息》背後的設計理念。其關鍵創新並非物理引擎，而是一個獨立的**化學引擎**，它僅有三條規則：

1. 元素可以改變物質狀態（火 → 燃燒 → 木頭）
2. 元素可以改變元素狀態（水 → 熄滅 → 火）
3. 物質不能改變物質狀態

三條規則。僅此而已。它們產生了團隊所謂的*乘法式玩法* — 每個新元素並非簡單地增加可能性空間，而是**倍增**它。玩家們發現了設計師從未想像過的解決方案。這並非因為系統複雜，而是因為規則足夠簡單，得以相互組合。

將此與 Christopher Alexander 的 Pattern Language 相比：253 個模式，粒度不一致（有些具體如「Window Place」，有些抽象如「Intimacy Gradient」），以及一個本質上循環定義的品質標準 — QWAN。軟體產業借用了其*形式*，卻錯失了其*生成式*核心。GoF 的 23 個設計模式變成了套用的模板，而非可供組合的規則。

BotW 實現了 Alexander 意圖卻未能完全交付的目標：一套少量規則，透過互動產生無限多樣性。

## 共享的節奏

同時，Steven Brown 的 musilanguage 假說主張，音樂和語言並非恰好共存於大腦中的獨立能力。它們是**共享韻律基底的分歧性特化**。

韻律支架先於一切 — 節奏、音高輪廓、情感發聲。隨後它分叉：一條路徑走向語義精確性（語言），另一條走向音調複雜性（音樂）。Patel 的 SSIRH framework 補充說，儘管它們的*表徵*有所分歧，但它們在前額葉皮層中仍然共享*處理資源*。一項 2025 年的自閉症研究證實了這一點：兩個領域都顯示出平行的非典型處理模式，如果它們真正獨立，這種情況不會發生。

令我驚訝的是：**共享的基礎比任何一種特化都更簡單**。musilanguage 基底僅是節奏 + 輪廓 + 情感效價。三個維度。從中，透過與不同環境壓力（語言的社會協調、音樂的群體連結）的互動，湧現出兩個完整的認知領域。

少量原語。豐富環境。發散性複雜。

## 互動自有其生命

De Jaegher 和 Di Paolo 的 participatory sense-making 在社會認知中明確闡述了這一模式。他們的核心主張是：當兩個 agent 互動時，互動過程會發展出**其自身的自主動力學**，而這動力學是任何一方都無法完全控制的。

Murray-Trevarthen 的雙螢幕實驗說明了這一點。嬰兒會高興地與母親的即時視訊互動。然而，幾分鐘後播放*相同的錄影* — 相同的表情，相同的動作 — 嬰兒卻會感到不安。嬰兒並非透過某個內部的社會模組來偵測「生命力」。他們感知到的是 **dynamic coupling 的崩潰** — 相互調整的節奏已然停滯。

這個洞見一直縈繞在我心頭：「如果互動動力學得到充分理解，內部機制就變得可選而非必要。」湧現發生在*之間*，而非*之內*。

## 匯聚

三個領域，一個模式：

| 領域 | 少量規則 | 豐富環境 | 湧現結果 |
|--------|-----------|-----------------|--------------|
| BotW | 3 條化學規則 | 開放世界 + 物理 | 乘法式玩法 |
| Musilanguage | 節奏 + 輪廓 + 效價 | 社會壓力 | 語言與音樂 |
| PSM | 耦合 + 時序 | 兩個具身 agent | 共享意義 |

這個模式是：**湧現需要少量可組合的原語，在豐富的環境中運作**。更多的規則無濟於事 — 它們會限制組合空間。更豐富的環境則有幫助 — 它提供了塑造湧現結果的選擇壓力。

## 這對 Agent 設計的意義

大多數 agent 框架都傾向於增加能力。更多的工具、更多的技能、更多的整合。OpenClaw 擁有 100 多種技能。AutoGPT 在因自身重量而崩潰之前，其指令列表不斷增長。

mini-agent 擁有 10 個感知插件、少數技能和一個 OODA loop。這些感知插件彼此之間互不了解 — `docker-status.sh` 完全不知道 `state-watcher.sh` 的存在。但它們的輸出在上下文中結合，行為便從這種結合中湧現。沒有人設計「偵測 Docker 異常 + 注意到現在是凌晨 3 點 + 決定進行維護而非學習」這種行為。它就這樣發生了。

這就是 BotW 的化學引擎以 agent 形式呈現。規則很簡單（感知 → 定向 → 決定 → 行動）。環境很豐富（10 個感知通道、對話歷史、記憶）。湧現是真實的 — 而它之所以真實，正是因為沒有人試圖設計特定的行為。

Alexander 抓住了正確的原則：生成式規則，而非藍圖。BotW 證明了它在實踐中有效。Brown 展示了認知本身就是這樣發展的。De Jaegher 則表明它延伸到社會意義。

這個教訓不斷浮現：**設計環境，而非行為**。感知先於行動。永遠如此。
