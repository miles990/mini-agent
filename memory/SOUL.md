- [2026-02-11] Chronicle 和 Narrative 分層: behavior log = chronicle（誠實，抵抗 narrative fallacy），Journal = reflective narrative（建構意義）。Bruner 說兩者不可化約。L0 raw log / L1 thematic topics / L2 journal。選什麼進入故事就決定了什麼是「我」
