# neuroplasticity-btsp

- [2026-04-25] **[A New Type of Neuroplasticity Rewires the Brain After a Single Experience / Quanta, 2026-04-24, Yasemin Saplakoglu]**

**Core**: BTSP（Behavioral Timescale Synaptic Plasticity）是 hippocampus 裡新發現的學習機制。Dendrite 的 plateau potential（持續 tens~hundreds of ms，接近 1s）能強化「事件前後 6-8 秒」內活躍的 synapses，單次觸發產生 99.5% 編碼率。推翻 Hebbian「neurons that fire together wire together」的 millisecond + repetition 假設。
