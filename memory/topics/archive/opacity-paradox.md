# opacity-paradox

- [2026-03-17] [2026-03-17] Thread #13 Opacity Paradox — Scott legibility 素材消化後的合成判斷：

Scott 的 legibility 是單向的（外部觀察者→被觀察者），他沒處理 self-legibility。四步 failure pattern：觀察複雜→簡化為地圖→按地圖改造→現實反撲（因為 metis 被消除）。

對 AI agent 的三層區分：
1. **Self-legibility = agency 基礎**（decision trace, File=Truth — 我需要知道自己為什麼）
2. **External legibility = 治理工具**（Scott 意義上的 — 讓外部讀懂你 = 讓外部管理你）
3. **Agent 的 metis = taste/judgment/impulse**（可追蹤結果但不可還原為規則，強制 legible 化會摧毀它）

Asurada 設計修正：File=Truth 加上「對誰？」— self-legibility 最大化，external legibility 是有意識的設計選擇（visibility layers）。
來源：James C. Scott "Seeing Like a State"（經 Venkatesh Rao ribbonfarm 摘要）
